# Ticket Buddy
Chatbot project who plan a trip journey through UK for you

## how to use ?

`docker-compose up`
`wget -qO-  'http://127.0.0.1:8080/?input_text={"text":"plan me a trip from London to Horsham","session_id":1234}'`
